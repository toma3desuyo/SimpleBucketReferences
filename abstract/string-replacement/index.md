## 概要
文字列置換は、文字列中の一部を特定の文字列やパターンに基づいて別の文字列へ差し替える操作である。単純置換と正規表現置換が代表的。

## 基本例
```pseudo
入力 = "color"
結果 = 入力.replace("or", "our")  # "colour"
```
- 最初の一致だけかすべての一致かをオプションで選ぶ。

## 応用パターン
### 正規表現置換
```pseudo
パターン = /(\d{4})(\d{4})(\d{4})(\d{4})/
整形 = パターン.replace(カード番号, "$1-$2-$3-$4")
```
- キャプチャグループを再利用し、複雑な整形を行う。

### 条件付き置換
```pseudo
結果 = replace(入力, 条件: 無名関数(一致):
    return 一致.toUpperCase()
)
```
- 一致文字列に応じて置換内容を動的に決定する。

## 注意点
1. **イミュータブル性**: 多くの言語で文字列は不変。置換は新しい文字列を返すためメモリ使用量を考慮する。
2. **繰り返し置換**: 連続で置換すると意図しない再置換が起こる。パターンを工夫するか一括処理する。
3. **ロケール**: 大文字小文字変換を伴う置換はロケール依存の結果になる場合がある。
