# リストのソート

## 概要
コレクション要素を順序付けし直す操作。データの見やすさや探索効率向上に使う。

## 振る舞い
比較関数を渡せば任意の基準で並べ替えられる。安定ソートなら同値要素の元の順序が保持される。典型計算量は O(n log n)。インプレースか非破壊かは実装次第。

## 例
function sort(list, compare):
  runs = detect_runs(list)
  return merge_runs(runs, compare)


## 注意点
数値を文字列比較しないよう型を揃えた比較関数を渡す。大規模データはメインスレッドを長時間ブロックしないよう分割や並列化を検討。
