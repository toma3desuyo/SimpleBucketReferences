## 概要
除算演算子は 2 つの数値を割り算し、商を得る。整数除算と浮動小数点除算で挙動が異なり、ゼロ除算や丸め規則を設計に反映させる必要がある。

## 特徴
- **商の型**: 整数同士でも結果を浮動小数にする言語と、切り捨てる言語がある。
- **ゼロ除算**: 例外を送出するか特別な値（∞、NaN）を返すかを仕様で定義する。
- **精度**: 浮動小数点では誤差が発生する。多倍長が必要なら専用ライブラリを使用。

## 基本例
```pseudo
平均 = 合計 / 件数
比例配分 = 金額 / レート
```
- 小さな値で割るとオーバーフローする可能性がある。単位を揃えてから計算する。

## 応用パターン
### 整数除算と余り
```pseudo
商 = 17 / 5  # 3
余り = 17 % 5  # 2
```
- 余り演算と組み合わせてループ処理や周期計算に用いる。

### 正規化
```pseudo
ベクトル = (x, y, z)
長さ = sqrt(x*x + y*y + z*z)
正規化 = (x/長さ, y/長さ, z/長さ)
```
- 分母がゼロになるケースを防ぐため、長さが 0 の場合のフォールバックを用意する。

## 注意点
1. **桁落ち**: 大きな数値を大きな数値で割ると精度が失われる。スケールを調整する。
2. **符号の扱い**: 負数を含む除算では商や余りの符号規則が言語で異なる。仕様を確認する。
3. **性能**: 除算は乗算よりコストが高い。繰り返し計算する場合は逆数を事前に求めて乗算に置き換えることもある。
