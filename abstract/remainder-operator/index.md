## 概要
余り演算子（mod、%）は、整数除算の結果として余った値を返す。周期的な処理やインデックスの循環、ハッシュ計算などに使用される。

## 特徴
- **結果の範囲**: 0 以上、除数未満であることが一般的。負の数を扱う場合の符号は言語仕様による。
- **計算コスト**: 除算と同等。繰り返し利用する場合はビット演算で代替できるケースもある。
- **用途の多様性**: 循環バッファ、ハッシュテーブル、奇偶判定など。

## 基本例
```pseudo
週番号 = 日数 % 7
偶数か = (値 % 2) == 0
```

## 応用パターン
### 循環インデックス
```pseudo
index = (index + 1) % 配列.長さ
```
- インデックスが末尾に達したら先頭へ戻るリングバッファを実現する。

### ハッシュ
```pseudo
バケット = ハッシュ(キー) % バケット数
```
- 余りを利用して配列インデックスに割り当てる。

## 注意点
1. **負数の扱い**: `-1 % 5` が 4 になる言語と -1 になる言語がある。期待する挙動に合わせて正規化する。
2. **性能**: 除数が 2 の冪ならビットマスク `(値 & (n-1))` で代用できる。
3. **浮動小数点**: 一部言語では小数に対しても余り演算が定義されている。丸め誤差で結果が不正確になる場合がある。
