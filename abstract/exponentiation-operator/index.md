## 概要
累乗演算子は `a^b` や `a**b` の形で、基数 `a` を `b` 回乗じた結果を計算する。指数法則を利用した計算や指数関数的な成長を表現する。

## 特徴
- **指数の型**: 整数指数と実数指数で意味が異なる。実数指数は対数を用いて計算される。
- **負の指数**: 1/(a^|b|) を表し、小数結果になる。
- **優先順位**: 乗算より高い優先順位を持ち、右結合である実装が多い。

## 基本例
```pseudo
面積 = 半径^2 * π
減衰 = 基準 * (0.5 ^ 経過回数)
```
- 指数は変数や式で指定できる。整数指数なら繰り返し乗算でも実装できる。

## 応用パターン
### 累乗の高速化
```pseudo
関数 累乗(基数, 指数):
    もし 指数 == 0 なら return 1
    もし 指数 偶数 なら:
        半分 = 累乗(基数, 指数/2)
        return 半分 * 半分
    else:
        return 基数 * 累乗(基数, 指数-1)
```
- 二乗法で計算量を O(log n) に抑える。

### 指数スケール
```pseudo
音量 = 基準音量 * 10^(デシベル/20)
```
- 対数単位を線形値に変換する場面で使う。

## 注意点
1. **オーバーフロー**: 指数が大きいとすぐに型の上限を超える。結果をログで保存するなどスケールを調整する。
2. **精度誤差**: 浮動小数点指数は内部で対数計算するため精度に限界がある。必要なら任意精度演算を使う。
3. **負数の偶奇**: 負の基数と分数指数の組み合わせは複素数を生む。実数範囲で制限する場合は入力を検証する。
